Basis Git Commando's

Hier zijn vijf veelgebruikte Git-commando’s met uitleg:

  • git init – Start een nieuw Git-repository in de huidige map.
  • git clone <url> – Maakt een kopie van een bestaande repository.
  • git add <bestand> – Voegt bestanden toe aan de staging area (klaar om te committen).
  • git commit -m "bericht" – Slaat de veranderingen op in de repository met een beschrijvend bericht.
  • git push – Stuurt je commits naar een externe repository (bijvoorbeeld GitHub).
